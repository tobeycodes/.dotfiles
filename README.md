# Dotfiles

Public dotfiles repository.

## Requirements

- macOS with Homebrew
- Arch Linux with Pacman and Paru

## Installation

```bash
git clone git@github.com:tobeycodes/.dotfiles.git ~/dotfiles
cd ~/dotfiles
brew bundle install # macOS
pacman -S --needed - < .config/pacman/packages.txt # Arch Linux
stow .
```
